## 🚀 The Journey to Greatness
Welcome to the exciting world of cricket analytics! This project is a data-driven journey to discover the absolute **Best 11 Cricketers** in the world. We are not just looking for good players; we are hunting for legends who can dominate the game! 🌟

Our mission is to assemble a team so balanced and powerful that they can:
- 🏏 **Score Big**: Average at least **180 runs**.
- 🛡️ **Defend Tight**: Defend targets of **150 runs**.

Success is just around the corner, and with this analysis, we are one step closer to cricket perfection!

## 🌟 The Blueprint for Champions
We have defined strict and ambitious criteria to filter the best from the rest. Here is the formula for our Dream Team:

### 💥 Openers: The Firestarters
*Setting the stage with explosive starts!*
| Parameter        | Description                      | Criteria |
| ---------------- | -------------------------------- | -------- |
| Batting Average  | Average runs scored per innings  | > 30     |
| Strike Rate      | Runs scored per 100 balls        | > 140    |
| Innings Batted   | Total innings played             | > 3      |
| Boundary %       | Percentage of runs in boundaries | > 50     |
| Batting Position | Order in the batting lineup      | < 4      |

### ⚓ Middle Order: The Unshakable Anchors
*Stabilizing the innings and building momentum!*
| Parameter        | Description                     | Criteria |
| ---------------- | ------------------------------- | -------- |
| Batting Average  | Average runs scored per innings | > 40     |
| Strike Rate      | Runs scored per 100 balls       | > 125    |
| Innings Batted   | Total innings played            | > 3      |
| Avg. Balls Faced | Average balls faced per innings | > 20     |
| Batting Position | Order in the batting lineup     | > 2      |

### ⚡ Finishers: The Game Changers
*Finishing strong and turning the tide!*
| Parameter        | Description                     | Criteria |
| ---------------- | ------------------------------- | -------- |
| Batting Average  | Average runs scored per innings | > 25     |
| Strike Rate      | Runs scored per 100 balls       | > 130    |
| Innings Batted   | Total innings played            | > 3      |
| Avg. Balls Faced | Average balls faced per innings | > 12     |
| Batting Position | Order in the batting lineup     | > 4      |

### 🌪️ All-Rounders: The Ultimate MVPs
